Trouver son identifiant

Avec la mise en place de New Panthera, les anciens utilisateurs devrons se connecter avec leur identifiant et mot passe. Avant cette mise à jour il fallait se connecter avec son adresse mail. Nous allons vous montrer comment retrouver votre identifiant/pseudo que vous avez choisi au préalable lors de votre inscription.

Connectez-vous sur ATG SYSTEM si vous avez un compte GOLD, connectez-vous sur ATC SYSTEM si vous avez un compte CRYPTO.

Entrez les mêmes informations de connexion que vous utilisiez sur l’ancien site de PantheraTrade. À savoir votre email et votre mot de passe.

Appuyez sur l’avatar en haut à droite. Vous y trouverez votre pseudo/identifiant que vous avez choisi lors de votre inscription sur Panthera.

Connexion au nouveau back-office

Certains utilisateurs n’ont pas reçu par mail leurs informations de connexion sur New Panthera. À ce jour, toutes les données n’ont pas encore migrées sur le nouveau site. Nous allons simplement vous montrer comment récupérer vos informations de connexions.

Connectez-vous sur la page de réinstallation du mot de passe de NewPanthera.

Indiquez votre pseudo puis cliquez sur Submit.

Vous recevez par mail votre Login-ID ainsi que votre mot de passe.

Rendez-vous sur le back-office de Panthera pour y rentrer vos informations de connexion et accéder au site.

Vous pouvez également entrer votre adresse mail pour recevoir vos identifiants par email via cette page.

Effectuer son KYC

Pour faire le KYC (Vérification d’identité), cliquez sur UPDATE KYC en bas du menu de gauche.

Téléchargez votre document, carte d’identité/Passeport/Permis et recopiez le numéro de votre pièce d’identité dans la case ID NUMBER.

La photo ne doit pas dépasser 1Mo, utilisez un réducteur d’image sur Google si elle est trop grosse, IMG2GO est très bien comme site.

Votre KYC est en attente de validation.

Achat du voucher

Depuis le 18 mars 2022, vous devez payer un code Voucher de 50 $ afin d’activer vos adresses de retraits.

Dans le menu de gauche, cliquez sur Profile Vouchers.

Cliquez sur Purchase New Voucher.

Choisissez la crypto avec laquelle vous souhaitez payer, puis appuyer sur Click to Proceed.

Vous avez dans Deposit Address l’adresse ou vous devez envoyer les fonds. N’oubliez pas d’inclure les frais d’envoi de votre exchange.

Vous avez juste au-dessus le montant exact que l’adresse doit recevoir. Ici 49,97014 USDT.

En cliquant sur View Payment Status, vous pouvez suivre l’évolution de la transaction.

Le statut passera à Complété lorsque les fonds auront été reçus et confirmés plusieurs fois par le réseau.

Activation du voucher

Allez de nouveau sur Profile Vouchers.

Vous pouvez ainsi récupérer le code dans la colonne VOUCHER.

Cliquez ensuite sur Update Profile, en haut à gauche dans le menu.

Indiquez vos deux adresses de retraits : Votre adresse USDT au réseau ERC20 et votre adresse USDT au réseau TRC20 . Cliquez ensuite sur Click to Continue.

Dans OTP received in your Email, indiquez le code que vous avez reçu par email.

Dans la case en dessous insérer votre voucher code.

Cliquez sur Validate and submit.

Vos adresses de retraits ont bien été mis à jour.

ATTENTION : Vos adresses de retraits sont strictement personnelles ! Si vous souhaitez changer ces adresses, vous devez repayer un nouveau Voucher code (50 $). Ainsi, personne ne pourra détourner des fonds de votre compte.

Effectuer un retrait du compte MT4

Dans le menu à gauche, à la section Copytrade accounts, cliquez sur MT4 Withdrawal.

Cliquez maintenant sur MT4 New Withdrawal.

Dans Select MT4 Account : Sélectionnez le compte MT4 ou vous souhaitez effectuer un retrait.

Vous pouvez observer dans Available Balance les fonds que vous avez sur le compte choisi.

Indiquez dans Amount you want to Withdraw le montant que vous souhaitez retirer du compte MT4.

Cliquez sur Continue for verification.

Pour confirmer la transaction, indiquez le code OTP reçu par email.

Puis cliquez sur Validate and confirm OTP.

Effectuer un retrait du compte Panthera

Afin d’effectuer un retrait du compte Panthera, vos fonds doivent être sur le Wallet. Vous devez également avoir indiqué vos adresses de retraits en ayant acheté un voucher code.

Dans Select Payment mode indiquez l’adresse de retrait que vous souhaitez utiliser.

Dans Enter withdrawal amount indiquez la somme que vous souhaitez retirer. Vous pouvez retirer au minimum 10 $ et au maximum 100 $ (temporairement en attendant le retour à la normale avec 2000 $) par retrait.

Cliquez sur Continue for verification.

Pour confirmer la transaction, indiquez le code OTP reçu par email.

Puis cliquez sur Validate and confirm OTP.

Votre demande de retrait est effectuée. Vous recevrez une notification par email une fois celle-ci effectuée. En re-cliquant sur Withdraw Wallet vous pourrez suivre l’évolution du processus.

Effectuer un dépôt sur le Wallet

Pour effectuer un dépôt sur Panthera, cliquez sur Place New Request.

Sélectionnez la crypto monnaie de votre choix dans Select Purchase Option et indiquer dans Enter USDT amount you want to buy le montant que vous souhaitez déposer. Entre 10 et 100 000$.

Cliquez sur Click to Proceed.

Vous avez dans Deposit Address : l’adresse USDT TRC20 ou vous devez envoyer les fonds.

Juste au-dessus le montant exact que doit recevoir cette adresse, ici 1049,5665 USDT.

N’oubliez pas les frais de réseau en envoyant les fonds.

Vous pouvez consulter l’évolution de la transaction en cliquant sur View Payment Status.

Le statut passera à Complété lorsque les fonds auront été reçus et confirmés plusieurs fois par le réseau.

Effectuer un dépôt sur le compte MT4

Vos fonds sont arrivés sur le Wallet , vous pouvez les voir sur le Dashboard, il ne reste plus qu’à les transférer sur votre compte MT4.

Dans le menu à gauche, sous Copytrade Accounts cliquez sur MT4 Deposit.

Cliquez ensuite sur MT4 New Deposit.

Dans Select MT4 Account choisissez le compte ou vous souhaitez déposer vos fonds.

Indiquez dans Enter Deposit Amount la somme que vous souhaitez déposer. Minimum 50 $.

Appuyez sur Continue.

Votre dépôt est effectué.

Vous pouvez voir vos fonds sur votre compte MT4.

Inscription sur Coinstore

Pour bénéficier de l’AirDrop de la cryptomonnaie Legocoin suite à l’achat du Voucher sur PantheraTrade, vous devez créer un compte sur Coinstore avec la même adresse mail que vous utilisez sur PantheraTrade.

Ceux qui ont acheté un Voucher vont recevoir 2500 Legocoin soit environ 50$. Les autres vont recevoir 100 Legocoin soit approximativement 2$.

Pour s’inscrire sur Coinstore en bénéficiant d’avantages exclusifs, cliquez sur le lien ci-dessous.

S'inscrire sur Coinstore

Dans Email : Inscrivez la même adresse mail que celle associée à votre compte Panthera.

Dans Password : Créez votre mot de passe.

Dans Invitation code : Vérifiez que le code H95v6h est présent, sinon inscrivez-le manuellement.

Pour finir, acceptez les termes d’utilisations et appuyez sur Register.

Vous allez recevoir un code à 6 chiffres dans votre boîte mail. Inscrivez-le et passez à l’étape suivante.

Pour les personnes qui utilisent déjà la double authentification sur d’autres applications, vous pouvez l’activer sur Coinstore. Si vous n’êtes pas familier avec les 2FA, appuyez sur Cancel.

Afin de valider votre KYC, au-dessous de KYC Verification cliquez sur Go KYC.

Dans Country/Region : Indiquez votre lieu de résidence.

Dans Name : Indiquez votre Nom et Prénom.

Dans ID number : Indiquez le numéro de la pièce d’identité que vous souhaitez utiliser.

Pour finir, cliquez sur Submit.

Pour finaliser la deuxième étape de la validation du KYC, cliquez sur Advanced KYC.

Dans Photo Identification : Insérez une photo de votre pièce d’identité.

Dans KYC Portrait : Vous devez vous prendre en photo en tenant dans vos mains :

– La pièce d’identité ou vous avez précédemment indiqué son numéro.

– Une feuille blanche contenant :

Votre numéro d’utilisateur Coinstore.(Écrivez : UID : ‘Votre numéro’ ).

Suivez le tutoriel ci-dessous pour retrouver son UID.

La date du jour.(Écrivez : Date : ‘MOIS/JOUR/2022’ ).

Votre Nom et Prénom écrit en majuscule avec une signature au-dessous.(Écrivez : Signature : ‘ Votre Nom Prénom’).

Lorsque tout est bon, appuyez sur Submit.

Vous pouvez trouver votre UID en cliquant sur l’avatar en haut à droite de Order.

Transférer les commissions sur le Fund Wallet

Les gains d’affiliations (rebates) s’affichent pour l’instant dans les Commission Wallet.

Pour les transférer sur le Fund Wallet : rendez-vous dans le menu puis cliquez sur Commission Transactions.

Les fonds disponibles dans le Commission Wallet sont affichés dans Commission balance.

Indiquez dans Enter amount to Deposit la somme que vous souhaitez transférer.

Attention : Vous ne pouvez uniquement entrer qu’un nombre entier. Ici 215.68 n’est pas valable.

Ensuite cliquez sur Continue for verification.

Pour confirmer la transaction, indiquez le code OTP reçu par email.

Puis cliquez sur Confirm details and submit.

Le transfert a été finalisé avec succès.

Les fonds sont maintenant sur le Fund Wallet.

L’affiliation sur AutoTrade

Depuis le 15 novembre 2021, l’affiliation est ouverte à tous les utilisateurs d’AutoTrade avec différentes positions pour les plus valeureux.

Pour trouver votre lien de parrainage, connectez-vous sur le back-office de Panthera .

Dans votre Dashboard, glissez vers le bas jusqu’à la section Referral Link, vous y verrez votre lien.

Au niveau des rémunérations, elles tombent dans le Commission Wallet situé à droite du Fund Wallet sur le back-office de Panthera.

Nous allons vous expliquer le plan de rémunération et chacune des positions que vous pouvez aller chercher.

Rang n°1 : TRADER

Toutes personnes s’inscrivant sur Panthera débute en tant que Trader (client). Il peut ainsi parrainer d’autres clients pour toucher une rémunération sur leurs trades.

À chaque trade opéré sur le compte de vos clients, vous allez gagner 2$/lots tradé. Les lots correspondent au volume affiché à côté de sell ou buy sur vos compte MetaTrader 4 (MT4). 1 lot équivaut à 10,000$ de capital.

Si vous parrainez un client qui dépose 1000$ et que le robot effectue un trade dans la journée, il aura eu un lot de 0.1 et vous allez gagner 0,2$. Si le robot fait deux trades dans la journée, vous aller gagner 0,2$ x2 soit 0,4$ ainsi de suite. Plus vous allez référer des clients, plus le volume de vos équipes va augmenter, plus votre rémunération journalière va augmenter.

Un Trader gagne donc 2$/lots sur son premier niveau (parrainage direct) et 1$/lots sur son deuxième niveau (les parrainages de vos clients).

Le Trader pourra obtenir une meilleure rémunération en passant la position supérieure : Sub IB Partner.

Rang n°2 : SUB IB PARTNER

Pour devenir SIP (Sub IB Partner) vous devez avoir un minimum de 3000$ sur votre compte, avec 20 clients inscrits personnellement (avec dépôt effectué et licence achetée) et avoir un volume des dépôts de vos clients d’au moins 30 000$.

Le Sub Ib Partner touchera 3$/lots sur son propre compte, le Trader n’est pas éligible à cette rémunération. Si vous avez 10 000$ sur votre compte de trading, le robot ouvrira une position avec un lot de 1. Vous allez gagner à chaque trade 3$ grâce à votre propre compte.

La rémunération sur les clients change également et diffère en fonction des positions des filleuls.

Le SIP touchera 5$/lots sur ses parrainages directs qui ont le statut de Trader. Si votre filleul direct passe la postion de SIP comme vous, vous toucherez alors plus 5$/lots mais 2$/lots.

Deuxième cas : Si votre filleul à le statut de trader et que celui-ci parraine un autre Trader, vous allez gagner 4$/lots sur le client de votre filleul. Cependant, si votre filleul à une position de SIP, vous toucherez uniquement 1$/lots sur toutes les personnes de votre deuxième niveau. Vous pouvez retrouver tous les cas de figure sur le tableau ci-dessus.

Rang n° 3 : SUB IB

Pour devenir SIB (Sub IB) vous devez avoir un minimum de 7000$ sur votre compte, avec 35 clients inscrits personnellement (avec dépôt effectué et licence achetée) et avoir un volume des dépôts de vos clients d’au moins 75 000$.

Le Sub Ib touchera 5$/lots sur son propre compte. Si vous avez 10 000$ sur votre compte de trading, le robot ouvrira une position avec un lot de 1. Vous allez gagner à chaque trade 5$ grâce à votre propre compte.

La rémunération sur les clients change également et diffère en fonction des positions des filleuls.

Le SIB touchera 7$/lots sur ses parrainages directs qui ont le statut de Trader. Si votre filleul direct passe la postion de SIP, vous toucherez alors plus 7$/lots, mais 4$/lots. Si votre filleul direct passe la position de Sub IB (SIB) alors que vous êtes toujours SIB, vous toucherez uniquement 2$ lots également.

Deuxième cas : Si votre filleul à le statut de trader et que celui-ci parraine un autre Trader, vous allez gagner 6$/lots sur le client de votre filleul. En fonction de l’évolution des positions de chacun, la rémunération change. Vous pouvez retrouver tous les cas de figure sur le tableau ci-dessus.

Rang n°4 : IB

Pour devenir IB vous devez avoir un minimum de 15 000$ sur votre compte, avec 50 clients inscrits personnellement (avec dépôt effectué et licence achetée) et avoir un volume des dépôts de vos clients d’au moins 150 000$.

L’IB touchera 7$/lots sur son propre compte. Si vous avez 20 000$ sur votre compte de trading, le robot ouvrira une position avec un lot de 2. Vous allez gagner à chaque trade 14$ grâce à votre propre compte.

La rémunération sur les clients change également et diffère en fonction des positions des filleuls.

L’IB touchera 9$/lots sur ses parrainages directs qui ont le statut de Trader. Si votre filleul direct passe la position de SIP, vous toucherez alors plus 9$/lots, mais 6$/lots. Si votre filleul direct passe la position d’IB alors que vous êtes toujours IB, vous toucherez uniquement 2$ lots également.

Deuxième cas : Si votre filleul à le statut de trader et que celui-ci parraine un autre Trader, vous allez gagner 8$/lots sur le client de votre filleul. En fonction de l’évolution des positions de chacun, la rémunération change. Vous pouvez retrouver tous les cas de figure sur le tableau ci-dessus.

Rang n° 5 : MIB

Pour devenir MIB vous devez avoir un minimum de 250 000$ sur votre compte, avec 100 clients inscrits personnellement (avec dépôt effectué et licence achetée) et avoir un volume des dépôts de vos clients d’au moins 500 000$. Vous devez également avoir un volume d’équipe de 5 000 000$ et 2500 membres dans vos équipes.

Le MIB touchera 8$/lots sur son propre compte. Si vous avez 250 000$ sur votre compte de trading, le robot ouvrira une position avec un lot de 25. Vous allez gagner à chaque trade 200$ grâce à votre propre compte.

La rémunération sur les clients change également et diffère en fonction des positions des filleuls.

Le MIB touchera 10$/lots sur ses parrainages directs qui ont le statut de Trader. Si votre filleul direct passe la position de SIP, vous toucherez alors plus 10$/lots, mais 7$/lots. Si votre filleul direct passe la position de MIB comme vous, vous toucherez uniquement 2$ lots également.

Deuxième cas : Si votre filleul à le statut de trader et que celui-ci parraine un autre Trader, vous allez gagner 9$/lots sur le client de votre filleul. En fonction de l’évolution des positions de chacun, la rémunération change. Vous pouvez retrouver tous les cas de figure sur le tableau ci-dessus. Et d’autre complément avec le tableau-ci dessous.

Cas pratique

Quels sont les rémunérations d’un Trader ?

Un trader peut gagner des rebates de deux manières uniquement :

En référent directement un client, ce qui lui fera gagner 2$/lots sur chaque trade.

Si son client dirrect réfère un autre client, cela lui fera gagner 1$/lots sur le client de son filieul direct.

Vous avez parrainé Bastien qui a déposé 1000$ sur son compte, vous gagnez 2$/lots sur chaque trade effectué sur le compte de Bastien. Bastien parraine Guillaume qui a déposé 10000$ sur son compte, vous gagnez 1$/lot sur chaque trade opéré sur le compte de Guillaume. Sur cette journée le robot a effectué un seul trade avec un gain de 1%, Bastien avec son capital de 1000$ a eu un lot de 0.1 sur son compte et un gain de 10$. On rappelle que 1 lot = 10,000$ de capital. Guillaume quant à lui eu un lot de 1 avec un gain de 100$. Sur Bastien vous gagnez 2$/lots soit 2$ × 0.1(le lot sur le compte de Bastien) = 0.2. Vous avez gagné 0.2$ avec le capital de Bastien. Pour Guillaume : 1$/lot × 1(le lot sur le compte de Guillaume) = 1$. Vous avez gagné 1$ avec le capital de Guillaume. Par contre, si Guillaume réfère Miriam, vous ne gagnerez pas de commissions sur elle en tant que Trader.

À titre d’exemple, si vous étiez SIP (SUB IB PARTNER) vous gagnerez 5$/lots sur Bastien, 4$/lots sur Guillaume et 3$/lots sur Miriam.

En tant que Sub IB (SIB) vous gagnez 7$/lots sur Bastien, 6$/lots sur Guillaume et 5$/lots sur Miriam.

En tant qu’IB vous gagnez 9$/lots sur Bastien, 8$/lots sur Guillaume et 7$/lots sur Miriam.

En tant que MIB vous gagnez 10$/lots sur Bastien, 9$/lots sur Guillaume et 8$/lots sur Miriam.

Les licences

Vous pouvez obtenir une rémunération sur les licences de vos filleuls en fonction de votre position.

Pour cela il suffit d’envoyer son lien d’affiliation Pansaka que vous trouvez sur la Dashboard de Pansaka. Par exemple un Sub IB Partner touchera ≈ 7,5% de la licence s’il parraine un Trader alors qu’un IB touchera 15%.

Cas pratique : Si vous êtes IB et que votre filleul SIB parraine un Trader, vous gagnez ≈ 3,75%. Si ce même trader réfère un autre Trader, vous gagnez également ≈ 3,75%. Nous détaillons approximativement les gains sur les filleuls directs dans le tableau ci-dessous. Prenez en compte que ces taux varient fréquemment, mais deviennent très intéressants lorsque vous développer une grosse équipe.

L’affiliation est 100% optionnel, mais peut vous rapporter gros si vous êtes bien accompagné. Le support NewBusinessOnline est disponible 7j/7 pour ses affiliés.
