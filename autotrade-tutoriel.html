Informations et astuces sur AutoTrade

Bienvenue sur la page d’astuces et tutoriels vis-à-vis d’AutoTrade. Consultez toutes les sections de cette page qui sera actualisée chaque semaine pour connaître le fonctionnement et astuces sur AutoTrade en complément des pages déjà présentes : AutoTrade Gold et AutoTrade Crypto.

Trouver son identifiant

Avec la mise en place de New Panthera, les anciens utilisateurs devrons se connecter avec leur identifiant et mot passe. Avant cette mise à jour il fallait se connecter avec son adresse mail. Nous allons vous montrer comment retrouver votre identifiant/pseudo que vous avez choisi au préalable lors de votre inscription.

Connectez-vous sur ATG SYSTEM si vous avez un compte GOLD, connectez-vous sur ATC SYSTEM si vous avez un compte CRYPTO.

Entrez les mêmes informations de connexion que vous utilisiez sur l’ancien site de PantheraTrade. À savoir votre email et votre mot de passe.

Appuyez sur l’avatar en haut à droite. Vous y trouverez votre pseudo/identifiant que vous avez choisi lors de votre inscription sur Panthera.

Connexion au nouveau back-office

Certains utilisateurs n’ont pas reçu par mail leurs informations de connexion sur New Panthera. À ce jour, toutes les données n’ont pas encore migrées sur le nouveau site. Nous allons simplement vous montrer comment récupérer vos informations de connexions.

Connectez-vous sur la page de réinstallation du mot de passe de NewPanthera.

Indiquez votre pseudo puis cliquez sur Submit.

Vous recevez par mail votre Login-ID ainsi que votre mot de passe.

Rendez-vous sur le back-office de Panthera pour y rentrer vos informations de connexion et accéder au site.

Vous pouvez également entrer votre adresse mail pour recevoir vos identifiants par email via cette page.

Effectuer son KYC

Pour faire le KYC (Vérification d’identité), cliquez sur UPDATE KYC en bas du menu de gauche.

Téléchargez votre document, carte d’identité/Passeport/Permis et recopiez le numéro de votre pièce d’identité dans la case ID NUMBER.

La photo ne doit pas dépasser 1Mo, utilisez un réducteur d’image sur Google si elle est trop grosse, IMG2GO est très bien comme site.

Votre KYC est en attente de validation.

Achat du voucher

Depuis le 18 mars 2022, vous devez payer un code Voucher de 50 $ afin d’activer vos adresses de retraits.

Dans le menu de gauche, cliquez sur Profile Vouchers.

Cliquez sur Purchase New Voucher.

Choisissez la crypto avec laquelle vous souhaitez payer, puis appuyer sur Click to Proceed.

Vous avez dans Deposit Address l’adresse ou vous devez envoyer les fonds. N’oubliez pas d’inclure les frais d’envoi de votre exchange.

Vous avez juste au-dessus le montant exact que l’adresse doit recevoir. Ici 49,97014 USDT.

En cliquant sur View Payment Status, vous pouvez suivre l’évolution de la transaction.

Le statut passera à Complété lorsque les fonds auront été reçus et confirmés plusieurs fois par le réseau.

Activation du voucher

Allez de nouveau sur Profile Vouchers.

Vous pouvez ainsi récupérer le code dans la colonne VOUCHER.

Cliquez ensuite sur Update Profile, en haut à gauche dans le menu.

Indiquez vos deux adresses de retraits : Votre adresse USDT au réseau ERC20 et votre adresse USDT au réseau TRC20 . Cliquez ensuite sur Click to Continue.

Dans OTP received in your Email, indiquez le code que vous avez reçu par email.

Dans la case en dessous insérer votre voucher code.

Cliquez sur Validate and submit.

Vos adresses de retraits ont bien été mis à jour.

ATTENTION : Vos adresses de retraits sont strictement personnelles ! Si vous souhaitez changer ces adresses, vous devez repayer un nouveau Voucher code (50 $). Ainsi, personne ne pourra détourner des fonds de votre compte.

Effectuer un retrait du compte MT4

Dans le menu à gauche, à la section Copytrade accounts, cliquez sur MT4 Withdrawal.

Cliquez maintenant sur MT4 New Withdrawal.

Dans Select MT4 Account : Sélectionnez le compte MT4 ou vous souhaitez effectuer un retrait.

Vous pouvez observer dans Available Balance les fonds que vous avez sur le compte choisi.

Indiquez dans Amount you want to Withdraw le montant que vous souhaitez retirer du compte MT4.

Cliquez sur Continue for verification.

Pour confirmer la transaction, indiquez le code OTP reçu par email.

Puis cliquez sur Validate and confirm OTP.

Effectuer un retrait du compte Panthera

Afin d’effectuer un retrait du compte Panthera, vos fonds doivent être sur le Wallet. Vous devez également avoir indiqué vos adresses de retraits en ayant acheté un voucher code.

Dans Select Payment mode indiquez l’adresse de retrait que vous souhaitez utiliser.

Dans Enter withdrawal amount indiquez la somme que vous souhaitez retirer. Vous pouvez retirer au minimum 10 $ et au maximum 100 $ (temporairement en attendant le retour à la normale avec 2000 $) par retrait.

Cliquez sur Continue for verification.

Pour confirmer la transaction, indiquez le code OTP reçu par email.

Puis cliquez sur Validate and confirm OTP.

Votre demande de retrait est effectuée. Vous recevrez une notification par email une fois celle-ci effectuée. En re-cliquant sur Withdraw Wallet vous pourrez suivre l’évolution du processus.
